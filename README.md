## 📝 笔记内容概览

### 1. 量子计算基础 (量子比特)
本部分详细记录了量子计算的入门知识，包括：
- **数学基础**：复数、线性代数、狄拉克符号系统。
- **量子比特 (Qubit)**：叠加态、Bloch 球面表示。
- **量子门操作**：单比特门 (X, Y, Z, H)、两比特门 (CNOT) 及其矩阵表示。
- **量子纠缠**：贝尔态 (Bell States) 的生成与性质。
- **量子测量**：投影测量与算符。

### 2. 量子机器学习应用 (Qbit+GBSVM)
本部分探讨了如何利用机器学习方法解决量子问题，主要包括：
- **两比特纠缠判别**：
    - 使用 **PPT (Positive Partial Transpose)** 判据作为标签生成依据。
    - 构建量子态的特征向量 (如 Pauli 期望值)。
    - 使用 **SVM (支持向量机)** 进行分类训练。
- **三比特纠缠研究**：
    - 生成随机纯态 (Haar 随机) 与全可分态。
    - 对比 **SVM** 与 **GBSVM (粒球支持向量机)** 的分类效果。
    - 特征选择与超参数优化 (GridSearchCV)。
    - 研究噪声 (物理信道噪声与采样噪声) 对模型性能的影响。
