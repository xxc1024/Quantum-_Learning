## 一 -- 正交测量
1. 辅助系统 (指针) $→$ 临时存储单元: 量子测量需要引入“指针”（辅助系统）来关联微观量子态和宏观可观测结果，如测量自旋态时，指针是粒子的位置（向上/向下）
2. 张量积$⊗$ $→$ 多维数组/矩阵克罗内克积: 系统与指针的联合状态用张量积表示（如$|\psi\rangle\otimes|0\rangle$），类比计算机中, 矩阵的克罗内克积（将两个系统的状态空间合并为更高维空间）
3. 幺正变换 U $→$ 线性变换/置换操作
	1. 线性变换（如矩阵乘法，保持数据的内积/距离不变）
	2. 置换操作（如排序，不改变数据的总量但重新排列状态）
	3. 关键：**幺正变换是可逆的**，不丢失信息。
4. 投影算符 $\{E_{\mathrm{a}}\} \to$ 筛选/分类器
	1. 筛选器: 从混合数据中提取符合条件的子集，如$E_\mathrm{a}\mid\psi\rangle$是$|\psi\rangle$在$E_\mathrm{a}$ 子空间的投影
	2. 分类器 (将输入状态分到不同类别，如测量指针得到结果$a$对应$E_\mathrm{a}$ 的输出)
5. 密度矩阵 $\rho \to$ 概率分布/统计状态
	1. 混合态用密度矩阵 $\rho = \sum E_{\mathrm{a}} \rho E_{\mathrm{a}}$ 表示
	2. 概率分布, 每个结果的概率由 $\rho$ 的对角元给出
	3. 统计状态, 如多个系统的平均状态
6. 量子测量的工程逻辑
	1. 输入: 被测系统初始态$|\psi\rangle+$指针初始态$|0\rangle$ (如空寄存器);
	2. 处理: 幺正变换 $U$ 将系统与指针关联 (如$U|\psi\rangle\otimes|0\rangle=\sum E_\mathrm{a}\left|\psi\rangle\otimes|a\rangle\right)$
	3. 输出: 测量指针得到结果$a$, 概率为$\langle \psi|E_{\mathrm{a}}|\psi\rangle$ (类比计算机中读取寄存器的值)
	4. 更新: 系统状态变为$\frac{E_\mathrm{a}\mid\psi\rangle}{||E_\mathrm{a}\mid\psi\rangle||}$ (若结果已知) 或混合态 $\sum E_\mathrm{a}\rho E_\mathrm{a}$ (若结果未知) 
7. 总结
	1. 量子电路中的测量: 量子比特的测量本质是投影算符操作，输出 0/1 的概率由量子态在基矢上的投影决定
	2. 量子算法设计: 如 Shor 算法中测量步骤**依赖投影算符提取因子信息**
	3. 误差分析: 标准量子极限 $\Delta x \geq \sqrt{ht/m}$ 类比计算机中传感器的噪声下限，指导测量装置的设计
	4. 量子信息处理：理解测量如何丢失信息（不可逆），帮助设计可逆量子算法
## 二 -- 广义测量
1. 广义测量与正交测量
	1. 正交测量
		1. 投影算符 $\{E_a\}, {E_a}^2=E_a$
		2. 测量后状态一定是正交的
		3. 两次连续测量结果相同
		4. 通过在特定基上测量
	2. 广义测量(POVM)
		1. 正算子值测度 $\{E_a\}, E_a\ge 0$
		2. 测量后状态不一定正交
		3. 两次连续测量结果可能不同
		4. 通过指针系统与系统纠缠实现
2. POVM可以看作是一种概率性条件操作: `if (rand()<p){ ... }else{ ... }`, 结果具有概率性
3. 从幺正相互作用到POVM
	1. 系统与指针纠缠：系统 $A$ 与指针 $B$ 通过幺正算符 $U$ 发生相互作用，初始态为 $|\psi\rangle_A \otimes |0\rangle_B$，演化后成为纠缠态：$$U: |\psi\rangle_A \otimes |0\rangle_B \mapsto \sum_a M_a |\psi\rangle_A \otimes |a\rangle_B.$$其中 $\{|a\rangle_B\}$ 是指针的一个正交基，$M_a$ 是作用在系统上的测量算符。

	2. 测量指针：对指针在基 $\{|a\rangle_B\}$ 上进行正交测量，得到结果 $a$ 的概率为 $\|M_a|\psi\rangle\|^2$，系统后测量态为 $\frac{M_a|\psi\rangle}{\|M_a|\psi\rangle\|}$

	3. 导出POVM：定义效应算符 $E_a = M_a^\dagger M_a$，则结果概率可写为 $\langle\psi|E_a|\psi\rangle$（对于混合态 $\rho$，概率为 $\text{tr}(\rho E_a)$）这些 $E_a$ 满足：
		1. 厄米性：$E_a = E_a^\dagger$,
		2. 正性：$E_a \geq 0$（本征值非负）
		3. 完备性：$\sum_a E_a = I.$
		
4. POVM重要性质
	1. 非正交性：POVM对应的后**测量态不一定正交**，**连续测量结果也可能不一致**(非可重复)
	2. 自由度：给定POVM $\{E_a\}$,测量算符$M_a$不是唯一的(例如$M_a=U_a\sqrt{E_a}$,其中$U_a$任意幺正), 因此后测量态可以人为调控
	3. 物理实现：任何POVM都可通过系统与辅助系统(指针)幺正纠缠后，对辅助系统进行正交测量来实现
