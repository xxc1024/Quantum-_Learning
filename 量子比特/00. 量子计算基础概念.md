## 0. 数学知识补充
### (1) 符号标记
- $$|0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$; $|1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$$
- $|+\rangle$ = $\frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)$
  $|-\rangle$ = $\frac{1}{\sqrt{2}}(|0\rangle - |1\rangle)$
- $|\psi\rangle$ 称为"ket"（列向量）; $\langle\psi|$ 称为"bra"（行向量）
- 本征值=特征值
	1. X的本征值：+1, -1；
	   本征向量：$\frac{1}{\sqrt{2}}(|0\rangle+|1\rangle)$, $\frac{1}{\sqrt{2}}(|0\rangle-|1\rangle)$
	2. Z的本征值：+1, -1；
	   本征向量：$|0\rangle, |1\rangle$
### (2) 性质
- 内积定义：$\langle\phi|\psi\rangle = \phi^\dagger\psi$（其中$\phi^\dagger$是$\phi$的共轭转置）
- 正交性：$\langle 0|1\rangle = 0, \langle 0|0\rangle = \langle 1|1\rangle = 1$
- 归一化条件：$|\alpha|^2 + |\beta|^2 = 1$
- $\langle\psi|$是$|\psi\rangle$的共轭转置
- $\alpha$和$\beta$的相位（幅角）影响量子干涉效应 **?**
- 内积的几何意义：$|\langle\phi|\psi\rangle|^2$表示从$|\psi\rangle$态测量到$|\phi\rangle$态的概率 **?**

### (3) 张量积
- 用于区分独立系统和纠缠系统: 纠缠态不能表示为两个独立量子态的张量积
- 矩阵张量积(向量是特殊的矩阵)
  $$
  \begin{aligned}X \otimes Y & =\left[\begin{array}{ll}0 & 1 \\1 & 0\end{array}\right] \otimes\left[\begin{array}{cc}0 & -i \\i & 0\end{array}\right] \\& =\left[\begin{array}{cc}0 \cdot\left[\begin{array}{cc}0 & -i \\i & 0\end{array}\right] & 1 \cdot\left[\begin{array}{cc}0 & -i \\i & 0\end{array}\right] \\1 \cdot\left[\begin{array}{cc}0 & -i \\i & 0\end{array}\right] & 0 \cdot\left[\begin{array}{cc}0 & -i \\i & 0\end{array}\right]\end{array}\right] \\& =\left[\begin{array}{cccc}0 & 0 & 0 & -i \\0 & 0 & i & 0 \\0 & -i & 0 & 0 \\i & 0 & 0 & 0\end{array}\right]\end{aligned}
  $$
- $|00\rangle=|0\rangle \otimes|0\rangle$
- 张量具有分配律: $(A+B)\otimes C = A\otimes C + B\otimes C$
### (4) 复数
1. 定义
	- 复数形式：$z = a + bi$，其中$a$是实部，$b$是虚部，$i^2 = -1$
	- 模长：$|z| = \sqrt{a^2 + b^2}$
	- 幅角：$\theta = \arctan(\frac{b}{a})$
2. 欧拉公式: $e^{i\theta} = \cos\theta + i \sin\theta$

## 2. 布洛赫球面
<img src="attachments/Pasted image 20251229141818.png" width="425" />
1. 北极点 = $|0\rangle$（经典态0）
   南极点 = $|1\rangle$（经典态1）
2. 赤道：均匀叠加的状态
	- $|+\rangle$态：位于 X 轴正方向，代表 $(|0\rangle+|1\rangle) / \sqrt{2}$
	- $|-\rangle$态：位于 X 轴负方向，代表 $(|0\rangle-|1\rangle) / \sqrt{2}$
	- 它们之间的区别在于**相位（Phase）**，这在几何上表现为它们在赤道上的不同位置
3. 极角  ($\theta$): 从 **Z 轴正方向**（北极）到状态向量的**夹角**, 它决定了测量时得到$|0\rangle$和$|1\rangle$的概率大小
    - 当点靠近北极（$\theta$接近 0）, 测量结果为$|0\rangle$的概率更高。
    - 当点靠近南极（$\theta$接近 1）, 测量结果为$|1\rangle$的概率更高。
4. 方位角 ($\phi$)：状态向量在 XY 平面上的投影与 **X 轴正方向的夹角**。它代表了量子态的**相对相位**。
5. 一个通用的量子比特状态(纯态)  可以用这两个角度表示为：
   $$
   |ψ⟩ = cos(\frac{θ}{2})|0⟩ + e^{(i\phi)}sin(\frac{θ}{2})|1⟩
   $$
6. 量子门操作相当于在布洛赫球面上旋转量子态
	1. X 门：可以看作绕 X 轴旋转 180°。它把北极的 $|0\rangle$ 转到南极的 $|1\rangle$，也把**南极转回北极**
	   <img src="attachments/Pasted image 20251229141035.png" width="200" />
	2. Z 门：可以看作绕 Z 轴旋转 180°。它不会改变箭头与 Z 轴的夹角，因此测量为 $|0\rangle$ 和 $|1\rangle$ 的概率保持不变，只是在赤道平面内“转动方向”，也就是**改变相位**
	   <img src="attachments/Pasted image 20251229141052.png" width="225" />
	3. Hadamard 门：可以理解为把 Z 轴上的极点“搬”到 X 轴上。例如，它把 $|0\rangle$ 旋转到 $|+\rangle$，把 $|1\rangle$ 旋转到 $|-\rangle$。从布洛赫球的视角看，就是把指向 Z 轴的箭头，翻转到赤道上的两个对称位置。
	   <img src="attachments/Pasted image 20251229141234.png" width="200" />

因此，当你在电路图里看到一串单比特门时，可以在脑海里把那根箭头想象成在球面上被牵引着不断旋转：每一个门，都是一次几何上的“转身”。
## 3. 单比特量子门
1. 量子门是**酉矩阵**——它们对量子态向量进行线性变换，同时保证概率的总和始终为 1
2. 酉矩阵: 
	1. 是复数版的正交矩阵
	2. 保证变换前后信号能量不变
	3. 酉矩阵的逆等于其共轭转置
	4. 共轭转置
		1. 转置: 把矩阵的行和列互换
		2. 共轭: 对复数的虚部取反 (比如 2+3i 变成 2-3i)
3. X门(量子非门): $|0\rangle \leftrightarrow |1\rangle$，相当于**绕**x轴旋转180°
   $$
   X = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}
   $$
4. Y门：绕y轴旋转180°
   $$
   Y = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix}
   $$
5. Z门：绕z轴旋转180°
   $$
   Z = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}
   $$
6. H门: 创造出**均匀叠加态**
   $$
   H = \frac{1}{\sqrt{2}} \begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}
   $$
	1. 对 $|0\rangle$ 施加 H 门，会得到一个“一半是 $|0\rangle$，一半是 $|1\rangle$”的状态，记作 $|+\rangle$。 
	   $$
	   H|0\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle) = |+\rangle
	   $$
	2. 对 $|1\rangle$ 施加 H 门，会得到一个相位不同的叠加态，记作 $|-\rangle$。  
	   $$
	   H|1\rangle = \frac{1}{\sqrt{2}}(|0\rangle - |1\rangle) = |-\rangle
	   $$
	3. 经过 H 门之后，量子比特不再是一个确定的 0 或 1，而是同时拥有了两种可能性。这是量子算法能够并行处理大量信息的基础

## 4. 两比特量子门
1. CNOT门 (受控非门): 创造“纠缠”的工具
	1. 工作逻辑是
		1. 如果**控制比特**是 $|0\rangle$，那么**目标比特**保持不变
		2. 如果**控制比特**是 $|1\rangle$，那么对**目标比特**执行一个 X 门（翻转）
	2. 例子
	   <img src="attachments/Pasted image 20251229145337.png" width="325" />
		1. 基本约定：
			- 时间从左向右流动，最左侧的门最先应用
			- 线表示量子比特，符号表示量子门
			- 量子门的输入数等于输出数（量子运算可逆）
			- 受控门：黑色圆圈表示控制量子比特，十字表示目标量子比特
		2. 初始状态：$|00\rangle$（两个量子比特都是 $|0\rangle$, **前者是控制比特, 后者是目标比特**）
		3. 施加 H 门：第一个量子比特变成叠加态，整体状态为 $\frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)|0\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |10\rangle)$
		4. 施加 CNOT 门：
			1. 对于 $|00\rangle$ 部分，控制比特处于 $|0\rangle$，目标比特保持 $|0\rangle$ 不变，结果仍是 $|00\rangle$
			2. 对于 $|10\rangle$ 部分，控制比特处于 $|1\rangle$，目标比特从 $|0\rangle$ 翻转为 $|1\rangle$，结果变为 $|11\rangle$
		5. 最终，我们得到的状态是 $\frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$。这就是著名的贝尔态 (Bell State)，一个完美的纠缠态
		6. 在这个状态下，两个量子比特不再独立。如果你测量第一个是 0，那么第二个必然是 0；如果你测量第一个是 1，那么第二个必然是 1。它们被“纠缠”在了一起
2. 贝尔态 $\frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$ 
	1. 它同时具备: 
		1. 局部: 完全随机
		2. 整体: 完全相关
	2. 具体来说，如果我们做很多次实验，并记录两个比特的测量结果，会得到这样的统计
		1. 单独看第一个比特
			1. $P($第一个是$0) = 0.5$ 
			2. $P($第一个是$1) = 0.5$
		2. 单独看第二个比特：
			1. $P($第二个是$0) = 0.5$  
			2. $P($第二个是$1) = 0.5$
		3. 看“成对”的结果（联合分布）
			1. $P(00) = 0.5$ 
			2. $P(11) = 0.5$ 
			3. $P(01) = 0$
			4. $P(10) = 0$
	  <img src="attachments/Pasted image 20251229145840.png" width="600" />
	3. 也就是说，每个比特单独看都像是抛硬币一样随机，但两者永远一致，要么一起是 0，要么一起是 1。这就是“局部随机但整体高度相关”的纠缠特性   
	4. 这类纠缠态是许多量子协议（如量子密钥分发、超密编码等）的资源基础，也是量子算法中让“**比特不再独立计数，而是成指数级增长的整体**”的关键原因之一
	5. pyqpanda3生成贝尔态
``` python
# 导入必要的模块
from pyqpanda3.core import H, CNOT, draw_qprog, measure, QProg, CPUQVM

prog = QProg(2)  # 创建一个包含 2 个量子比特的量子程序

prog << H(0)  # 添加Hadamard门到量子比特0

prog << CNOT(0, 1)  # 添加CNOT门，控制量子比特0，目标量子比特1

prog << measure(0, 0) << measure(1, 1)  # 添加测量操作，将量子比特0、1的测量结果存储到经典比特0、1中

print(draw_qprog(prog))  # 打印量子程序的图形表示


machine = CPUQVM()  # 创建一个CPU量子虚拟机实例

shots = 4096  # 设置测量次数

machine.run(prog, shots)  # 运行量子程序，进行测量

result = machine.result().get_counts()  # 获取测量结果

print("测量结果：", result)  # 打印测量结果

  

p_00 = result.get('00') / shots  # 获取测量结果'00'的概率

p_11 = result.get('11') / shots  # 获取测量结果'11'的概率

print("测量结果'00'的概率：", p_00)

print("测量结果'11'的概率：", p_11)
```

4. **贝尔态**：
   - 两比特最大纠缠态：
     - $|B_1\rangle = (|00\rangle + |11\rangle)/\sqrt{2}$
     - $|B_2\rangle = (|00\rangle - |11\rangle)/\sqrt{2}$
     - $|B_3\rangle = (|01\rangle + |10\rangle)/\sqrt{2}$
     - $|B_4\rangle = (|01\rangle - |10\rangle)/\sqrt{2}$
## 5. 测量 
1. 坍缩: 测量会使量子态从叠加态坍缩到一个确定的状态, 例如$|\psi\rangle = \alpha|0\rangle + \beta|1\rangle \to$ 测量后变为$|0\rangle$ 或 $|1\rangle$
2. 概率: 得到0的概率 = $|\alpha|^2$，得到1的概率 = $|\beta|^2$
3. 由于量子计算的概率性，通常需要多次重复运行同一个算法，以统计出概率最高的那个结果，这个结果就是我们想要的答案
4. 非破坏性: 如果量子态已经是$|0\rangle$或$|1\rangle$，测量不会改变状态
5. 相位无关: 测量结果与全局相位无关（$\alpha \to -\alpha$, $\beta \to -\beta$不影响概率）
6. **无克隆定理: 未知量子态无法被精确复制**
