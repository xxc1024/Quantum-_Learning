## 一 -- EPR 对的不可分离
1. 最大纠缠态 (Bell 态)：
	1. 经典中，两人各持半把钥匙，单独无法开锁，但合起来可打开
	   量子中，Bell 态如$$|\phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$就是这种“关联钥匙”
		1. 单独测 Alice 的量子比特 A：约化密度矩阵 $$\rho_A = \frac{1}{2}I$$（完全随机，像抛硬币）
		2. 联合测 A 和 Bob 的 B：可 100% 确定两人结果的关联（如 A=0 则 B=0，A=1 则 B=1）
		3. 关键：纠缠态的信息是分布式隐藏的 -- 本地无法提取，需联合操作才能恢复。
	2. 局域操作与联合操作
		1. 局域操作：Alice 对 A 施加$σ_3$（翻转相位）或 $σ_1$（翻转比特），仅改变纠缠态的“局部标记”（如相位位 / 奇偶位），但不破坏整体关联（类比经典中本地修改文件的部分字段）
		2. 联合操作：需两人将量子比特放在一起，做 Bell 基测量（对应$σ_1⊗σ_1$和$σ_3⊗σ_3$的同时测量），才能提取全部信息（类比经典中分布式系统的联合计算）
2. 量子电路基础
	1. Hadamard 门（H 门）：
		1. 基变换的“转换器”
		2. 数学本质：幺正变换（可逆），将 $Z$ 基 ($|0⟩, |1⟩$) 转成 X 基 $(|+⟩ = \frac{1}{\sqrt 2}(|0⟩ + |1⟩), |−⟩ =\frac{1}{\sqrt 2}(|0⟩ − |1⟩)$；
		3. 关键性质：
			1. $H^2 = I$（两次变换回到原基）
			2. $Hσ_1H = σ_3$（X 基和 Z 基互转）——相当于“视角切换”。
	2. CNOT 门：可逆的“异或操作”
		1. 定义：两比特操作，控制位 $a$ 决定是否翻转目标位 $b$：$|a, b\rangle \rightarrow |a, a \oplus b\rangle$;
		2. 类比：经典异或门，但量子 CNOT 是可逆的（因幺正变换可逆）-- 经典异或无法从输出和一个输入恢复另一个输入，量子则可；
		3. 核心作用：生成纠缠——当控制位是叠加态时，CNOT 会让目标位与控制位纠缠（如 $H|0\rangle \otimes |0\rangle$ 经 CNOT 变成 $|\phi^+\rangle$）
	3. H+CNOT 组合：纠缠态的“生成器 / 解码器”
		1. 正向：标准基（$|00\rangle, |01\rangle, |10\rangle, |11\rangle$）→ Bell 基（四个最大纠缠态）；
		2. 反向：Bell 基→标准基（因 H 和 CNOT 都是可逆的）；
		3. 意义：这是量子通信（如超密编码、隐形传态）的基础电路——相当于“打包”经典信息到纠缠态，或“解包”纠缠态中的信息。

##  二 -- 量子纠缠
 1. 可分混态：没有量子纠缠的混态
	1. 可分混态是指AB两系统的关联完全是经典的，可以通过“各自状态的经典概率组合”表示。
	2. 两种等价定义
		1. 形式1 (乘积纯态系综): 混态$\rho_{AB}$可写成多个乘积纯态的加权平均：$$\rho_{AB} = \sum_{i} p_{i} \left( |\alpha_{i}\rangle\langle\alpha_{i}|_{A} \otimes |\beta_{i}\rangle\langle\beta_{i}|_{B} \right)$$其中 $p_{i} \geq 0$ 且 $\sum_{i} p_{i} = 1$，$|\alpha_{i}\rangle$是 $A$ 的纯态，$|\beta_{i}\rangle$ 是 $B$ 的纯态。
		2. 形式2 (乘积密度算符加权和)：更一般地，可扩展到乘积混态的组合：$$\rho_{AB} = \sum_{i,j} p_{ij} \rho_{A,i} \otimes \rho_{B,j}$$$\rho_{A,i}$是 $A$ 的混态，$\rho_{B,j}$是 $B$ 的混态，$p_{ij}$是经典联合概率
	3. 可分性不依赖 “具体表示方式”，**只要存在至少一种表示是乘积态的组合即可**。例如：两比特均匀随机态 $ρ = \frac{1}{4} I \otimes I$, 既可以写成 4 个乘积纯态 $(|00⟩, |01⟩, |10⟩, |11⟩)$ 的平均，也可以写成 4 个最大纠缠态（Bell 态）的平均 —— 但只要有第一种乘积表示，它就是可分的。

2. 纠缠混态：存在量子关联的混态
	1. 如果一个混态**无法写成任何乘积态的组合**，则称为不可分混态（纠缠混态）
	2. 经典关联是 “两个独立随机变量的联合分布”（如抛两枚独立硬币），量子关联是 “无法分解为独立变量的联合分布”（如 Bell 态的统计关联，测量 A 会瞬间影响 B 的结果）。

3. LOCC 操作：不能创造纠缠的 “安全操作”
	1. Alice/Bob 只能在各自系统上做局域量子操作（如测量、幺正变换）；
	2. 可以交换经典信息（如测量结果），但不能传递量子比特。
	3. LOCC 的核心性质：
		1. 不能创造纠缠：如果初始态是可分的，LOCC 操作后依然是可分的（无法生成量子关联）
		   原因：局域操作和经典通信只能传递经典信息，无法引入量子纠缠的“超距关联”。
		2. 能制备所有可分态：例如要制备形式 1 的可分态：Alice 按概率 $p_i$ 制备 $|\alpha_i\rangle$，通过经典通信告诉 Bob 选 $|\beta_i\rangle$ 制备，最终联合态就是乘积纯态的加权平均。
## 三 -- PPT判别
1. 可分态: 二部量子态 $\rho_{AB}$（比如 Alice 和 Bob 各持一个量子比特）可分，等价于它能写成两个独立子系统态的加权组合：$$\rho_{AB} = \sum_{i} p_{i} \rho_{A}^{i} \otimes \rho_{B}^{i}$$（$p_{i}$ 是概率，$\rho_{A}^{i}$、$\rho_{B}^{i}$ 是各自子系统的密度矩阵）。
2. 部分转置: 
	1. 对矩阵 $M$，转置后 $M^{T}[i][j] = M[j][i]$（二维数组的行和列互换）。
	2. 对二部系统 $AB$，部分转置**仅对其中一个子系统（比如 B）做转置**，另一个子系统（A）保持不变：$$\rho^{PT} = I_{A} \otimes T_{B}(\rho_{AB})$$
	3. 举例：双量子比特态的密度矩阵是 4×4 矩阵，部分转置相当于把 B 对应的维度互换（比如 $\rho[a,b;c,d] \rightarrow \rho[a,d;c,b]$ ，其中 a/c 是 A 的维度，b/d 是 B 的维度）。
3. PPT 判据：可分性的 “快速筛选器”
	1. 判据内容: 如果 $\rho_{AB}$ 可分，则其部分转置 $\rho^{PT}$ 一定是正定矩阵（所有特征值≥0）
	2. 正定等价于 “二次型非负” 或 “特征值全非负”，是数值计算中容易验证的性质（比如用特征值分解算法）。
	3. 快速排除纠缠态：若 $\rho^{PT}$ 有负特征值→$\rho_{AB}$ 一定不可分（纠缠态）。
	4. 双量子比特的完全判据：对 2×2 系统（双量子比特），PPT 是可分的充要条件→若 $\rho^{PT}$ 正定→$\rho_{AB}$ 可分。

4. Werner 态的 PPT 判断
	1. Werner 态是双量子比特混合态：$\rho(\lambda) = \lambda|\Phi^+\rangle\langle\Phi^+| + \frac{1-\lambda}{4}I$
	2. $|\Phi^{+}\rangle$是最大纠缠态，$I$是单位矩阵，$\lambda$是纠缠成分的权重。
	3. 步骤 
		1. 计算部分转置$\rho(\lambda)^{PT} = \frac{1+\lambda}{4}I - \lambda|\Psi^{-}\rangle\langle\Psi^{-}|$
		2. 分析特征值$|\Psi^{-}\rangle$对应的特征值是$-\lambda$，其他特征值是$\frac{1+\lambda}{4}$
			1. 当$\lambda > \frac{1}{3}$时：$-\lambda < -\frac{1}{3}$，而$\frac{1+\lambda}{4} < \frac{1+\frac{1}{2}}{4} = \frac{3}{8}$（假设$\lambda < 1$）$\rightarrow$存在负特征值$\rightarrow\rho(\lambda)$不可分
			2. 当$\lambda \leq \frac{1}{3}$时：所有特征值$\geq 0$ $\rightarrow \rho(\lambda)$可分（双量子比特充要条件）
